# Lamad Content Graph Data

This directory contains the structured content node data extracted from the `/docs` directory for use in the Lamad learning platform.

## Files

### content-nodes.json (4.2 MB)

Generated JSON file containing the complete content graph:
- **430 total nodes** extracted from documentation
- **6 epics**: Autonomous Entity, Governance, Governance Layers, Public Observer, Social Medium, Value Scanner
- **61 personas**: User roles across all epics
- **260 scenarios**: Concrete use cases (.feature files)
- **Reference materials**: 91 organizations, 4 books, 4 videos, 2 audio, 1 article, 1 document
- **Taxonomy**: Complete categorization and navigation structure
- **3 suggested learning paths**: Curated journeys through the content
- **Relationships**: Interconnected nodes based on hierarchy, tags, and paths

## Generation

This file is generated by running:

cd elohim-app

## Regeneration

To regenerate this file after updating `/docs`:

1. Make changes to markdown or feature files in `/docs`
2. Run `node scripts/extract-docs-to-nodes.js`
3. New `content-nodes.json` will be generated with updated content
4. Commit the updated JSON file

## Notes

- File is tracked in git for easy deployment
- File size is ~4.2 MB (acceptable for web delivery with gzip)
- Consider lazy loading or chunking if size grows beyond 10 MB
- All content is embedded in the JSON (not loaded separately)
- Node IDs are deterministic (same file = same ID across regenerations)
